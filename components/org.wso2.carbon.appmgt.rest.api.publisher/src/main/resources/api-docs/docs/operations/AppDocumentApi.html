APPDocumentApi

Get App Documents

Get a list of documents belonging to an App.

URL

http://apis.wso2.com/api/appm/publisher/v1.1/apps/{appType}/id/{appId}/docs

HTTP Method

GET

Scope

appm:read

CURL Example

curl -H "Authorization: Bearer b0982cd2aacd463ff5f63cd5ebe58f4a" "http://localhost:9763/api/appm/publisher/v1.1/apps/webapp/id/0950b09c-0afe-47f2-88d6-72e9b88b1931/docs"

Sample Response

HTTP/1.1 200 OK Content-Type: application/json { "previous": "", "list": [ { "sourceType": "INLINE", "sourceUrl": null, "otherTypeName": null, "documentId": "0bcb7f05-599d-4e1a-adce-5cb89bfe58d5", "summary": "This is a sample documentation for v1.0.0", "name": "PhoneVerification App Documentation", "type": "HOWTO" }, { "sourceType": "URL", "sourceUrl": "http://wiki.cdyne.com/index.php/Phone_Verification", "otherTypeName": null, "documentId": "4145df31-04f1-440c-8d08-68952874622c", "summary": "This is the URL for online documentation", "name": "Online Documentation", "type": "SAMPLES" } ], "next": "", "count": 2 }
